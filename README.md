# 動画広告をぶっ飛ばすChrome拡張
YouTubeやニコニコ動画などの広告を16倍速で再生し、素早くスキップできるChrome拡張機能です。

## インストール方法

### 1. リポジトリをダウンロード
- GitHub のリポジトリページで [Code] → [Download ZIP] をクリック
- ダウンロードしたZIPファイルを解凍

### 2. Chrome に拡張機能を読み込む
1. Chrome を開き、アドレスバーに `chrome://extensions/` と入力
2. 右上の **デベロッパーモード** をオン
3. **パッケージ化されていない拡張機能を読み込む** をクリック
4. 解凍したフォルダを選択

### 3. 拡張機能をピン留め
- ブラウザのアドレスバー右側の拡張機能アイコンをクリック
- 「じーのスキップ」の横のピンアイコンをクリック → ツールバーに固定されます

## 使い方

### 基本操作
- 対応サイト（YouTube、ニコニコ動画など）を開く
- **S キー** を押すと動画/広告が16倍速で再生されます
- もう一度 **S キー** を押すか、3秒経過すると自動で通常速度に戻ります（3秒ルール）
- ポップアップ右上の **Go!/Stop! ボタン** でも加速を開始/停止できます（Sキーを使わなくてもOK）

### 設定
1. ツールバーのじーのスキップアイコンをクリック
2. 以下の設定が可能です：
   - **加速キー**：デフォルトはSキー。任意のキーに変更可能
   - **対応サイトURL**：YouTubeやニコニコ以外のサイトを追加可能

設定後は **タブを更新** してから有効になります。
